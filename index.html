Sobre Nicotech

Tecnología pensada para resolver problemas reales.

En Nicotech Software creemos que un buen sistema no solo debe funcionar, sino también ser claro, escalable y fácil de mantener.

Trabajamos junto a empresas, instituciones públicas y emprendedores, acompañando cada etapa del proyecto: desde la definición de los requerimientos hasta la implementación, capacitación y soporte. Entendemos que detrás de cada sistema existen personas, procesos y objetivos que merecen una solución confiable. Proyectos

Algunos de nuestros trabajos

Sistema de Gestión Social

Sistema web desarrollado para la Municipalidad de Rawson destinado a gestionar la entrega de ayudas sociales de manera segura y organizada. Permite administrar personas, grupos familiares, usuarios, secretarías y tipos de ayuda, registrar entregas con auditoría, generar comprobantes en PDF, enviarlos por correo electrónico, consultar historiales y obtener estadísticas para mejorar el seguimiento, la transparencia y la toma de decisiones en la gestión social.

Django Python PostgreSQL Docker Nginx

Playón de Secuestro y Guarda de Vehiculos Municipal

Sistema web desarrollado para optimizar la gestión del Playón de Secuestro y Guarda de Vehiculos Municipal, permitiendo registrar ingresos y egresos de vehículos, administrar la ocupación de los espacios disponibles y mantener un historial completo de movimientos. La plataforma facilita el trabajo del personal mediante una interfaz intuitiva, mejorando el control, la trazabilidad y la organización de las operaciones diarias del playón.

Django Python Docker PostgreSQL Nginx
